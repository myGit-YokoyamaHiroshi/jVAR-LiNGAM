# jVAR-LiNGAM
There are sample scripts of Jacobian-informed vector-autoregressive linear non-Gaussian model (j-VAR-LiNGAM) tha is developed to estimate the causally hierarchical directed network structures reflected in synchronous neural oscillation. 
<br><br>
